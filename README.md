# NH Web Design Portfolio Website

A professional, elegant, and blazing-fast single-page portfolio website for **NH Web Design** (`nhwebdesign.co.uk`). Built to be light, highly performant, and fully optimized for seamless deployment on **Cloudflare Pages**.

## 🚀 Features

- **Modern & Responsive Design:** Beautifully styled from scratch using **Tailwind CSS** following a clean White, Orange (`#f26522`), and Dark Slate Grey theme. Fully responsive across mobile, tablet, and desktop viewports.
- **Bespoke Services Grid:** Showcases 4 clear business pillars: Website Design, Graphic Design, Branding, and Web Application Development.
- **Transparent Pricing Packages:** Breaks down service offerings into three strategic tiers (**Starter**, **Growth**, and **Scale**) with comprehensive feature checklists targeted at different audiences, completely eliminating hidden fees.
- **Premium Monthly Care Plan:** Features an eye-catching, high-converting split-card layout highlighting the **£40/mo** support and maintenance ecosystem (Managed Hosting, Security, Backups, Performance Opt, and Content Edits).
- **Interactive Portfolio & Badges:** Restricted to live, custom-built projects featuring floating category badges (`Web App` for Snooker Scoreboard and `Website` for Preston Auto Care).
- **Mobile-Optimized Navigation:** Clean header navigation that transforms into a fully functional interactive mobile drop-down hamburger menu.
- **Production-Ready Contact Form:** Integrated with **Formspree** for frictionless backend email processing and secure, centralized **Cloudflare Turnstile** bot verification.
- **Smooth Scrolling & Animations:** Enhances user experience with professional scroll-driven interactions via **AOS (Animate On Scroll)**, while ensuring critical above-the-fold content loads instantly.
